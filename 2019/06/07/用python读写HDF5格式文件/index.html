HDF5介绍

       HDF(Hierarchical Data File)是美国国家高级计算应用中心(National Center for Supercomputing Application,NCSA)为了满足各种领域研究需求而研制的一种能高效存储和分发科学数据的新型数据格式 。
       HDF5适合存储大量的二进制信息，并且提供并行IO加快读写速度。
       我们可以用python的h5py包来读写HDF5文件。
